Wordle Game
Гра Wordle - це текстова гра, в якій гравець повинен вгадати слово за допомогою послідовного введення його літер.

Як грати
Запустіть програму.

Оберіть режим гри:

Режим 1: Слово дня - гравець повинен вгадати слово, яке обране автоматично на кожну гру або встановлене як "слово дня".
Режим 2: Випадкове слово - гравець повинен вгадати випадкове слово зі списку.
Якщо обраний Режим 1 (Слово дня), гравець буде грати із тим самим "словом дня" протягом всього дня, якщо вони вирішать грати знову.

Якщо обраний Режим 2 (Випадкове слово), гравець отримає нове слово для кожної гри.

Гравець вводить літери слова по одній, намагаючись вгадати його.

Гра продовжується до тих пір, поки гравець не вгадає слово або не вирішить припинити гру.

Вимоги
C++ Compiler: Для компіляції та запуску програми.

Запуск програми
Скомпілюйте програму за допомогою вашого компілятора C++.
Запустіть отриманий виконуваний файл.
Примітки
Гра автоматично визначає слово дня або обирає його випадковим чином для кожної нової гри.

Гравець може обирати, чи хоче він грати знову після завершення кожної гри.
